## Огляд

GeographyApp -- це Windows Forms застосунок, що дозволяє вести персональну базу географічних об'єктів. Дані організовані у чотири рівні ієрархії: материки містять країни, країни - регіони, регіони - міста. Подвійний клік на будь-якому записі миттєво фільтрує дочірні об'єкти. Кнопка «На карті» відкриває вбудоване вікно з Google Maps для вибраного місця.

## Можливості

**Об'єкти та їхні поля**

| Рівень | Поля |
|--------|------|
| Материк | Назва, населення, площа (км²) |
| Країна | Назва, материк, населення, площа, столиця, форма правління |
| Регіон | Назва, країна, населення, тип (область / штат / …), адм. центр |
| Місто | Назва, країна, регіон, населення, координати (широта / довгота) |

**Робота з даними**

Для кожного рівня доступні операції додавання, редагування та видалення. Усі форми перевіряють коректність введення перед збереженням і захищають від дублювання назв. Повідомлення про помилки з'являються всередині вікна, не закриваючи його.

**Пошук і навігація**

Рядок пошуку фільтрує записи поточного розділу за назвою в реальному часі. Подвійний клік на рядку таблиці переходить на рівень нижче і автоматично застосовує фільтр - наприклад, клік на материку показує лише країни цього материка.

**Сортування**

Будь-який розділ можна відсортувати за назвою, населенням або площею (де застосовно). Сортування враховує регістр і діакритичні знаки.

**Карта**

Кнопка «На карті» відкриває вбудоване вікно з Google Maps через Microsoft WebView2. Пошуковий запит формується автоматично з назви вибраного об'єкта.

## Встановлення

### Вимоги

- Windows 10 / 11
- [.NET 10 Runtime](https://dotnet.microsoft.com/download/dotnet/10.0)
- [Microsoft Edge WebView2 Runtime](https://developer.microsoft.com/microsoft-edge/webview2/) (зазвичай вже присутній на Windows 11)

### Запуск із вихідного коду

```bash
git clone https://github.com/Mykhailo-Berkuta/GeographyApp.git
cd GeographyApp
dotnet run --project GeographyApp/GeographyApp.csproj
```

### Збірка у виконуваний файл

```bash
dotnet publish GeographyApp/GeographyApp.csproj -c Release -r win-x64 --self-contained
```

Готовий `.exe` з'явиться у `GeographyApp/bin/Release/net10.0-windows/win-x64/publish/`.

## Структура проєкту

```
GeographyApp/
├── Forms/
│   ├── MainForm.cs          # Головне вікно: таблиця, пошук, сортування, навігація
│   ├── MapForm.cs           # Вбудована карта (WebView2 + Google Maps)
│   ├── ContinentForm.cs     # Форма додавання / редагування материка
│   ├── CountryForm.cs       # Форма додавання / редагування країни
│   ├── RegionForm.cs        # Форма додавання / редагування регіону
│   └── CityForm.cs          # Форма додавання / редагування міста
├── Models/
│   ├── GeographicObject.cs  # Абстрактний базовий клас (Name, Population, IComparable)
│   ├── Continent.cs
│   ├── Country.cs
│   ├── Region.cs
│   ├── City.cs
│   └── Comparers/
│       ├── GeographicObjectComparer.cs
│       ├── ContinentComparer.cs
│       ├── CountryComparer.cs
│       └── RegionComparer.cs
├── Services/
│   └── DataManager.cs       # Колекції, JSON-серіалізація, тестові дані
└── Program.cs
```

## Дані та збереження

Усі дані зберігаються локально у файлі `geography_data.json` в робочому каталозі програми. Збереження та завантаження доступні через меню або відбуваються автоматично. Якщо файл відсутній, програма завантажує вбудовані тестові дані: кілька материків, країн, регіонів і міст (зокрема Харків і Київ).

Структура файлу:

```json
{
  "Continents": [...],
  "Countries":  [...],
  "Regions":    [...],
  "Cities":     [...]
}
```
